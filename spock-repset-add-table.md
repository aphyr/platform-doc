## SYNOPSIS
    ./nodectl spock repset-add-table REPLICATION_SET TABLE DB <flags>
 
## DESCRIPTION
    Add a table to a replication set.
 
## POSITIONAL ARGUMENTS
    REPLICATION_SET
    TABLE
    DB
 
## FLAGS
    -c, --cols=COLS
        Type: Optional[]
        Default: None
    -p, --pg=PG
        Type: Optional[]
        Default: None
